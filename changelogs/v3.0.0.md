# v3.0.0 变更详情

> **发布日期**: 2026-03-10  
> **类型**: 🎉 重大重构（架构级变更）  
> **前置版本**: v2.13.0

---

## 📋 变更概述

v3.0.0 是对整个规范体系的**完全重写**，从 v2 的多文件分散架构迁移到**单文件权威规范（RULES.md）**架构，大幅降低认知负担和 Token 成本，同时保留了 v2 全部 22 条核心约束和经过验证的机制。

---

## 🎯 核心变更

### 1. 架构重设计：多文件分散 → 单文件权威规范

| 维度 | v2 | v3 |
|------|----|----|
| 规范入口 | 三层递进（入口~39行 → 速查~420行 → 完整~4000+行） | **RULES.md 单文件（~530行）** |
| 核心文件数 | 6+ 文件（copilot-instructions → QUICK-REFERENCE → CONSTRAINTS → README → workflows → templates） | **1 文件**（RULES.md） + 按需加载工作流/模板 |
| 交叉验证负担 | 高（6+ 文件互相引用，反复出 Bug，需 CROSS-VALIDATION.md 清单） | **极低**（单文件无跨文件同步问题） |
| Token 加载量 | ~460 行（日常）/ 4000+ 行（完整） | **~530 行**（一次全文读取，后续不重复读） |

### 2. 工作流合并：13 种独立工作流 → 6 种路由 + 子类型

| v2 工作流 | v3 路由 |
|---------|--------|
| 01-requirement-dev / 03-optimization / 05-refactoring / 06-database / 09-opensource-init | `dev`（含 5 种子类型 checklist） |
| 02-bug-fix / 07-security / 08-incident | `fix`（含 3 种子类型 checklist） |
| 04-research / 10-analysis | `analyze`（含 2 种子类型 checklist） |
| 11-self-audit | `audit` |
| — | `chat`（对话模式，内联于 §1 N10） |
| — | `resume`（任务恢复，内联于 §5 N11） |

### 3. 意图识别重设计：决策树 → 三问判断法

| v2 | v3 |
|----|----|
| `decision-tree.yaml` 配置文件（YAML 格式，多层嵌套） | **三问判断法**（§2，3 个语义问题，不依赖关键词匹配） |
| 关键词触发为主，4 层递进流程 | 语义理解为主，关键词仅作辅助 |

### 4. 模板精简：29 个 → 8 个

| v2 | v3 |
|----|----|
| `core/templates/core/`（完整模板） | 合并精简为 8 个模板 |
| `core/templates/lite/`（精简模板） | — |
| `core/templates/extended/`（扩展模板） | — |
| `core/templates/common/`（公共模板） | — |

v3 模板：`cp-checklist.md` · `memory-session.md` · `project-profile.md` · `report-analysis.md` · `report-audit.md` · `report-build.md` · `report-fix.md` · `requirement.md`

### 5. 全流程 Mermaid 流程图

v3 在 RULES.md 开头新增完整的 Mermaid 流程图，可视化展示 N01→N14 全部节点、分支、约束触发点，AI 和人类均可一目了然。

### 6. 目录结构重组

```text
v2 结构:                          v3 结构:
ai-dev-guidelines/                ai-dev-guidelines/
├── core/                         ├── version/
│   ├── workflows/ (13种)         │   ├── v2/          # v2 归档
│   ├── templates/ (29个)         │   │   ├── core/
│   ├── standards/ (9个)          │   │   └── docs/
│   ├── tools/ (7个)              │   └── v3/          # v3 活跃
│   ├── self-fix/                 │       ├── RULES.md # 唯一权威规范
│   ├── best-practices/           │       ├── workflows/ # 6种路由
│   ├── CONSTRAINTS.md            │       └── templates/ # 8个模板
│   ├── QUICK-REFERENCE.md        ├── changelogs/
│   ├── META.yaml                 ├── CHANGELOG.md
│   └── ...                       └── README.md
├── docs/
├── projects/                     → Workspace/projects/ (已迁移到工作目录)
└── changelogs/
```
