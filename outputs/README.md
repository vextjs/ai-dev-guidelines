# Outputs Directory

> AI 生成的文档输出目录

此目录由 AI 自动创建和管理，用于存储各项目的开发文档。

## 结构说明

```
outputs/
└── <project-name>/
    ├── requirements/      # 需求开发输出
    ├── bugs/             # Bug 修复输出
    ├── optimizations/    # 性能优化输出
    ├── research/         # 技术调研输出
    ├── refactoring/      # 架构重构输出
    ├── database/         # 数据库变更输出
    ├── security/         # 安全修复输出
    └── incidents/        # 事故复盘输出
```

## 使用说明

- 此目录会随着 AI 执行任务自动填充
- 每个项目独立管理
- 建议定期归档历史文档
