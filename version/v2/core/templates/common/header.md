# 文档头部模板

> 所有 AI 生成的文档**必须**使用此头部模板，包含必要的元信息。

---

## 🔴 必填字段规范

以下字段为**强制必填**，AI 生成文档时**禁止**留空或保留占位符：

| 字段 | 要求 | 示例 |
|------|------|------|
| **创建日期** | 🔴 必须替换为真实日期（ISO 格式） | `2026-02-26` |
| **最后更新** | 🔴 必须替换为真实日期（ISO 格式） | `2026-02-26` |
| **项目** | 🔴 必须填写真实项目名 | `vext` |
| **状态** | 🔴 必须选择一个状态 | `🟢 已完成` |

```yaml
❌ 绝对禁止:
  - 日期字段保留 "YYYY-MM-DD" 占位符
  - 日期字段留空
  - 使用非 ISO 格式日期（如 "2月26日"、"Feb 26"）

✅ 正确做法:
  - 使用当前真实日期，格式 YYYY-MM-DD
  - 初次创建时「创建日期」和「最后更新」相同
  - 后续修改时只更新「最后更新」
```

---

## 📋 标准头部模板

```markdown
# [文档标题]

> **项目**: [项目名称]
> **任务ID**: [任务ID]
> **创建日期**: [YYYY-MM-DD] ← 🔴 必须替换为真实日期
> **最后更新**: [YYYY-MM-DD] ← 🔴 必须替换为真实日期
> **负责人**: [AI / 姓名]
> **Agent**: [zed-copilot / webstorm-copilot / cursor / vscode-copilot / ...]
> **状态**: 🟡 进行中 / 🟢 已完成 / 🔴 已中止

---
```

---

## 🏷️ 状态说明

| 状态 | 符号 | 使用场景 |
|------|------|---------|
| 进行中 | 🟡 | 任务正在执行 |
| 已完成 | 🟢 | 任务已完成 |
| 已中止 | 🔴 | 任务被取消或中止 |
| 待开始 | ⚪ | 任务等待开始 |
| 草稿 | 📝 | 文档为草稿状态 |

---

## ⚙️ AI 执行规范

```yaml
生成文档时:
  1. 读取此模板
  2. 获取当前真实日期（通过工具或系统）
  3. 替换所有 [占位符] 为真实值
  4. 🔴 自检: 确认日期字段不包含 "YYYY" 字样
  5. 🔴 自检: 确认 Agent 字段已填写当前编辑器/AI 标识

日期获取优先级:
  1. 使用 now() 工具获取（最准确）
  2. 从用户上下文推断
  3. 🔴 无法确定时必须询问用户，禁止留占位符
```
