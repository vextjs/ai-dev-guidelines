## 核心目标速览

permia 是通用 Node.js 细粒度 RBAC 权限管理库，框架无关。支持三类资源权限控制：

| 资源类型 | 格式 | 示例 |
|---------|------|------|
| 数据库/集合级 | `db:<collection>` | `db:users` |
| 字段级 | `db:<collection>:<field>` | `db:users:email` |
| HTTP 接口级（含栏目/按钮）| `<METHOD>:<path>` | `GET:/api/users` |

1. 默认 **FileAdapter（磁盘 JSON）** 持久化，零依赖，生产可用
2. 接口权限对应真实 HTTP `METHOD:/path`（METHOD 大写）
3. `getPermissions` / `getResources` 返回完整权限集供跨框架接入
4. MonSQLizeAdapter 作为可选独立包，适合多实例部署
5. CJS + ESM 双出口，TypeScript 类型完整
