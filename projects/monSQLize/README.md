# monSQLize 项目

> **项目名称**: monSQLize  
> **项目描述**: 轻量级 MongoDB ORM，提供多级缓存、分布式事务（Saga）、统一表达式系统（122个操作符）、通用函数缓存，100% 兼容 MongoDB 原生驱动
