## 📋 代码风格

### 缩进和格式

- **缩进**: 4 空格
- **引号**: 单引号
- **分号**: 必须

### 命名规范

| 类型 | 规范 | 示例 |
|-----|------|------|
| Controller 文件 | PascalCase | `TripProjectController.ts` |
| Service 文件 | PascalCase | `TripProjectService.ts` |
| Model 文件 | snake_case | `trip_project.ts` |
| Route 文件 | snake_case | `trip_project.ts` |
| Enum 文件 | snake_case | `trip_strategy_agent.ts` |
| Validator 文件 | snake_case | `trip_project_validator.ts` |
| 类名 | PascalCase | `TripProjectController` |
| 变量/函数 | camelCase | `getUserSubscription` |
| 常量 | UPPER_SNAKE_CASE | `FREE_PROJECT_LIMIT` |
| 枚举值 | PascalCase | `TripProjectStatusEnum.Upcoming` |
| MongoDB 集合 | snake_case (复数) | `trip_projects` |
| 路由路径 | snake_case | `/home/trip_project` |

---

## 📝 示例

```typescript
// 变量命名（camelCase）
const projectTitle = 'My Trip';
const tripDayCount = 5;

// 常量命名（UPPER_SNAKE_CASE）
const FREE_PROJECT_LIMIT = 3;
const MAX_RETRY_COUNT = 3;

// 类命名（PascalCase）
export default class TripProjectService extends Service {
    // 方法命名（camelCase）
    async createTripProject(data: any) {
        const { ctx } = this;
        return await ctx.model.TripProject.create(data);
    }
}

// 文件命名
// Controller/Service: TripProjectController.ts, TripProjectService.ts
// Model/config: trip_project.ts, config.default.ts
```
