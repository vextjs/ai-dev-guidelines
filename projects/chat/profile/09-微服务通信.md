## 🌐 跨服务通信规范

### 使用方式

所有跨服务 HTTP 调用统一使用 `ctx.http` 方法（由 `httpHelper` 中间件提供）。

### 核心方法

| 方法 | 签名 | 说明 |
|------|------|------|
| `postJSON` | `(tag, url, body, init?)` | POST JSON 数据 |
| `getJSON` | `(tag, url, init?)` | GET 并解析 JSON |
| `fetchJSON` | `(tag, url, init?)` | 通用请求（推荐） |
| `getStream` | `(tag, url, init?)` | 获取流式响应 |
| `postForm` | `(tag, url, formData, init?)` | 提交表单数据 |

**🔴 重要**: 所有方法的第一个参数是 `tag`（日志标识），用于日志追踪。

### 示例

```typescript
export default class SomeService extends Service {
    async callExternalApi(userId: string) {
        const { ctx } = this;
        const microService: any = _.get(ctx.app.config.custom, 'microservices.payment');
        try {
            const result = await ctx.http.getJSON(
                'SomeService.callExternalApi',           // tag: 日志标识
                `${microService.url}/internal/some-api/${userId}`,
                { timeoutMs: 5000, retries: 1 }          // 配置选项
            );
            return result;
        } catch (error: any) {
            ctx.logger.error('[SomeService] 调用失败', { userId, error: error.message });
            if (error.code) throw error;
            throw ctx.dsl.error.create('EXTERNAL_API_FAILED');
        }
    }
}
```

### 配置选项

```typescript
{
    timeoutMs: 5000,      // 超时时间（毫秒），默认 15000
    retries: 1,           // 重试次数，默认 1
    backoff: {            // 退避策略
        baseMs: 300,
        factor: 2,
        jitter: true,
    },
    headers: {},          // 自定义请求头
}
```

## 📡 依赖的外部服务

| 服务 | 配置键 | 通信方式 | 用途 |
|-----|--------|---------|------|
| payment | `microservices.payment` | HTTP REST | 订阅查询、积分消耗 |
| mining_agent | `microservices.mining_agent` | HTTP REST | AI Agent（需求挖掘、复制等） |
| push | `microservices.push` | HTTP REST | 消息推送 |
| search | `microservices.search` | HTTP REST | 搜索服务 |

## 📡 暴露的内部接口

| 接口 | 方法 | 鉴权 | 用途 |
|-----|------|------|------|
| `/internal/trip/sell-config/:tripId` | GET | 无 | 查询行程售卖配置 |
| `/internal/trip/unlock` | POST | 无 | 解锁行程（payment 调用） |
| `/internal/trip/project/:projectId/generate-title` | POST | internalAuth | 异步生成项目标题 |
| `/internal/itinerary/generate` | POST | userAuth | 创建行程 |
| `/internal/itinerary/callback/progress` | POST | internalAuth | 行程生成进度回调 |
| `/internal/itinerary/ungenerate` | POST | userAuth | 取消行程生成 |
| `/internal/sse` | GET | userAuth | SSE 连接 |
| `/internal/customer/sse` | GET | accessCode | 客户端 SSE |
| `/internal/customer/sse/isOnline` | GET | accessCode | 检查客户端在线状态 |
| `/internal/notify` | POST | 无 | 客户端通知 |
