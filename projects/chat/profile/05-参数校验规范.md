## 📋 唯一推荐方式

所有新代码**必须**使用 `schema-dsl`，**禁止**使用 Joi 编写新的校验逻辑。

## 🚀 三层标准模式（Validator → Service → Controller）

### Step 1: Validator 层（参数校验）

```typescript
// app/validator/home/trip_project_validator.ts
import { BaseContextClass } from 'egg';

export default class TripProjectValidator extends BaseContextClass {
    async create() {
        const { ctx } = this;
        const schema = ctx.dsl({
            title: 'string:1-200',
            project_name: 'string:1-200',
            share_source: 'string',
        });
        return ctx.validateDsl(schema, 'body');
    }

    async detail() {
        const { ctx } = this;
        const schema = ctx.dsl({
            _id: 'objectId!',
        });
        return ctx.validateDsl(schema, 'params');
    }
}
```

### Step 2: Service 层（业务逻辑 + 错误处理）

```typescript
// app/service/trip/TripProjectService.ts
import { Service } from 'egg';

export default class TripProjectService extends Service {
    async createTripProject(data: any) {
        const { ctx } = this;
        try {
            const existing = await ctx.model.TripProject.findOne({
                title: data.title, del_flag: 0
            });
            if (existing) {
                ctx.dsl.error.throw('PROJECT_ALREADY_EXISTS');
            }
            return await ctx.model.TripProject.create(data);
        } catch (error: any) {
            if (error.code) throw error;
            throw ctx.dsl.error.create('PROJECT_CREATE_FAILED');
        }
    }
}
```

### Step 3: Controller 层（调用 Service + 响应）

```typescript
// app/controller/home/TripProjectController.ts
import { Controller } from 'egg';
import TripProjectValidator from 'validator/home/trip_project_validator';

export default class TripProjectController extends Controller {
    private readonly validator: TripProjectValidator;

    constructor(ctx: any) {
        super(ctx);
        this.validator = new TripProjectValidator(ctx);
    }

    async create() {
        const { ctx } = this;
        // 1. 参数校验（校验失败自动返回多语言错误响应）
        await this.validator.create();
        // 2. 获取参数
        const form = ctx.state.form;
        // 3. 调用 Service（不需要 try-catch）
        const result = await ctx.service.trip.tripProjectService.createTripProject(form);
        // 4. 成功响应
        ctx.success(result);
    }
}
```

## 📖 DSL 语法规则

### 基本格式

```
type:min-max[!]
```

### 类型

| 类型 | 说明 | 示例 |
|------|------|------|
| `string` | 字符串 | `string:2-50` |
| `int` | 整数 | `int:1-100` |
| `float` | 浮点数 | `float:0-` |
| `email` | 邮箱 | `email!` |
| `url` | URL | `url` |
| `objectId` | MongoDB ObjectId | `objectId!` |
| `array` | 数组 | `array` |
| `boolean` | 布尔值 | `boolean` |

### 必填标记

| 格式 | 说明 | 示例 |
|------|------|------|
| `!` | 必填字段 | `email!` |
| 省略 | 可选字段 | `string:2-50` |

### 常用场景

```typescript
// 分页查询
const schema = ctx.dsl({
    page: 'int:1-',
    pageSize: 'int:1-100',
    keyword: 'string:0-50',
});
await ctx.validateDsl(schema, 'query');

// 路径参数
const schema = ctx.dsl({
    projectId: 'string:24-24!',
});
await ctx.validateDsl(schema, 'params');

// 创建/更新
const schema = ctx.dsl({
    title: 'string:1-200!',
    description: 'string:0-1000',
    status: 'int:0-2',
});
await ctx.validateDsl(schema, 'body');
```

## ⚠️ 常见错误

### ❌ 错误 1: 业务错误使用 throw new Error()

```typescript
// ❌ 错误
if (!user) throw new Error('User not found');

// ✅ 正确
if (!user) ctx.dsl.error.throw('USER_NOT_FOUND');
```

### ❌ 错误 2: Controller 层手动 try-catch

```typescript
// ❌ 错误
async create() {
    try {
        const result = await ctx.service.xxx.create(data);
        ctx.success(result);
    } catch (error) {
        ctx.fail(error.message);
    }
}

// ✅ 正确（不需要 try-catch）
async create() {
    await this.validator.create();
    const result = await ctx.service.xxx.create(ctx.state.form);
    ctx.success(result);
}
```
