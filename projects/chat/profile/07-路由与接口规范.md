## 📋 接口分组与鉴权

| 路由前缀 | 分组 | 鉴权方式 | 说明 |
|---------|------|---------|------|
| `/home/*` | home | `userAuth` (JWT) | 用户端接口，需登录 |
| `/internal/*` | internal | `internalAuth` (Token) | 内部服务通信 |
| `/open/*` | open | IP 白名单 / 特定鉴权 | 开放接口 |
| `/admin/*` | admin | `userAuth` (admin JWT) | 管理后台 |
| `/ws/*` | ws | WebSocket 鉴权 | 实时通信 |

## 📁 路由文件组织

### 路由总入口

```typescript
// app/router.ts
import homeRoute from './routes/home';
import internal from './routes/internal';
import openRoute from './routes/open';
import adminRoute from './routes/admin';
import wsRoute from './routes/ws';
import scheduleRoute from './routes/schedule';

export default (app: Application) => {
    homeRoute(app);
    internal(app);
    openRoute(app);
    adminRoute(app);
    wsRoute(app);
    scheduleRoute(app);
};
```

## 🚀 路由定义方式

### home 路由（使用 egg-router-group 分组）

```typescript
// app/routes/home/trip_project.ts
import { Application } from 'egg';
import { RouterGroup } from 'egg-router-group';

export default (app: Application, groupRouter: RouterGroup) => {
    groupRouter.group({
        name: '行程项目管理',
        prefix: '/trip_project',
    }, (g: RouterGroup) => {
        const ctrl = app.controller.home.tripProjectController;
        g.post('/', ctrl.create);
        g.get('/', ctrl.index);
        g.get('/:_id', ctrl.detail);
        g.put('/:_id', ctrl.update);
        g.delete('/:_id', ctrl.delete);
    });
};
```

### internal 路由（扁平式）

```typescript
// app/routes/internal/index.ts
export default (app: Application) => {
    const { router } = app;

    router.group({
        prefix: '/internal',
    }, (sub: RouterGroup) => {
        const internalAuth = app.middleware.internalAuth();
        const trip = app.controller.internal.internalTripController;

        sub.get('/trip/sell-config/:tripId', trip.getSellConfig);
        sub.post('/trip/unlock', trip.unlockTrip);
        sub.post('/trip/project/:projectId/generate-title', internalAuth, trip.generateProjectTitle);
    });
};
```

## 🔐 鉴权配置

### 用户鉴权（userAuth）

```typescript
// 在路由分组的 index.ts 中配置
const userAuth = app.middleware.userAuth({ level: 'basic' });
router.group({
    prefix: '/home',
    middlewares: [userAuth],
}, (sub) => { ... });
```

### 内部通信鉴权（internalAuth）

```typescript
const internalAuth = app.middleware.internalAuth();
sub.post('/some-internal-api', internalAuth, controller.method);
```

### 访问码鉴权（accessCode）

```typescript
const accessCodeAuth = app.middleware.accessCode();
sub.get('/customer/sse', accessCodeAuth, controller.method);
```
