## 🤖 Agent 策略模式

### 调用流程

```
BaseChatService.run(strategyEnum, params)
    → AgentsFactory 根据 strategyEnum 选择 Agent
        → Agent 继承 BaseAbstractAgent
        → 调用 LLM（OpenAI / DeepSeek）
        → 返回结果
```

### 核心文件

| 文件 | 职责 |
|------|------|
| `service/ai/BaseChatService.ts` | AI 对话基础服务，统一入口 `run()` |
| `service/ai/agents/AgentsFactory.ts` | Agent 工厂，根据策略枚举选择具体 Agent |
| `service/ai/agents/BaseAbstractAgent.ts` | Agent 基类，定义通用生命周期 |
| `service/ai/ConversationService.ts` | 会话管理 |
| `service/ai/MessageService.ts` | 消息管理 |
| `service/ai/MemoryService.ts` | 记忆管理 |

### 使用方式

```typescript
import { TripAgentStrategyEnum } from 'enum/trip/trip_strategy_agent';

// 在 Service / Controller 中调用
const result = await ctx.service.ai.baseChatService.run(
    TripAgentStrategyEnum.TripNameAgent,
    {
        content: '原始文本',
        language: ctx.headers['language'],
    }
);
```

## 📦 核心 Agent 类型

| Agent 目录 | 用途 |
|-----------|------|
| `agents/trip/plan/` | 行程规划 Agent（TripNameAgent、行程生成等） |
| `agents/trip/chat/` | 聊天对话 Agent |
| `agents/trip/ru/` | 需求理解 Agent（需求挖掘、总结） |
| `agents/trip/packing/` | 打包建议 Agent |
| `agents/tool/` | 工具调用 Agent |
| `agents/AISearchAgent.ts` | AI 搜索 Agent |
| `agents/CommanderAgent.ts` | 指令调度 Agent |
| `agents/TripQuizAgent.ts` | 行程问答 Agent |

## ⚠️ 重要约定

- 💡 AI Agent 相关的耗时操作应**异步化**，避免阻塞用户请求
- 💡 可通过 SSE / 回调接口通知结果
- 💡 Agent 策略枚举统一定义在 `typings/enum/trip/trip_strategy_agent.ts`
