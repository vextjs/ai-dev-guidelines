## 📦 项目信息

### 基本信息

| 项目 | 说明 |
|-----|------|
| 项目名称 | chat |
| 项目描述 | AITA 智能旅行助手核心聊天服务，负责对话管理、行程规划、AI Agent 调度等 |
| 框架版本 | Egg.js ^3.17.5 |
| Node.js | ≥16.0.0 |
| 数据库 | MongoDB (egg-mongoose ^4.0.1), Redis (egg-redis ^2.6.0) |
| 配置中心 | Nacos ^2.6.0 |
| 部署方式 | Docker + egg-scripts 多环境部署 |

### 本地命令

- **启动服务**: `npm run dev`（基于 `{工作区根目录}/chat/` 目录）

## 📁 目录结构

```
chat/
├── app.ts                        # 应用启动钩子（AppBootHook）
├── agent.ts                      # Agent 进程
├── package.json
├── tsconfig.json
│
├── app/
│   ├── router.ts                 # 路由总入口
│   ├── controller/               # 控制器层（按接口分组）
│   │   ├── home/                 # 🔒 用户端接口（需登录鉴权）
│   │   ├── internal/             # 🔑 内部服务通信接口
│   │   ├── open/                 # 🌐 开放接口
│   │   ├── admin/                # 👤 管理后台接口
│   │   ├── ws/                   # 🔌 WebSocket 控制器
│   │   └── schedule/             # ⏰ 定时任务控制器
│   ├── service/                  # 服务层
│   │   ├── ai/                   # 🤖 AI 服务（核心）
│   │   ├── trip/                 # 🧳 行程服务
│   │   ├── ipe/                  # 🗺️ 行程规划引擎
│   │   ├── sharedb/              # 📝 实时协同编辑
│   │   └── ...
│   ├── model/                    # 数据模型（Mongoose Schema，70+ 模型）
│   ├── routes/                   # 路由定义（按分组）
│   ├── middleware/               # 中间件
│   ├── validator/                # 参数校验器（Validator 层）
│   └── utils/                    # 工具函数
│
├── config/                       # 配置文件
│   ├── config.default.ts         # 默认配置
│   ├── config.local.ts           # 本地开发
│   ├── config.sit.ts             # SIT 环境
│   ├── config.aita-uat.ts        # UAT 环境
│   ├── config.aita-prod.ts       # 生产环境
│   ├── plugin.ts                 # 插件配置
│   ├── dsl/                      # 🔴 schema-dsl 多语言错误码（唯一正确位置）
│   │   ├── index.ts
│   │   ├── zh.ts
│   │   ├── en.ts
│   │   └── hk.ts
│   ├── err_codes.ts              # ⚠️ 旧错误码（已废弃，不再新增）
│   └── err_codes_zh.ts           # ⚠️ 旧错误码（已废弃，不再新增）
│
├── typings/                      # TypeScript 类型定义
│   ├── enum/                     # 枚举（按业务域分目录）
│   ├── entity/
│   └── interface/
│
├── test/                         # 测试
├── scripts/                      # 运维脚本
├── docs/                         # API 文档
├── plans/                        # 需求计划
└── reports/                      # 诊断报告
```

### tsconfig 路径别名

| 别名 | 实际路径 | 用途 |
|-----|---------|------|
| `enum/*` | `typings/enum/*` | 枚举引用 |
| `interface/*` | `typings/interface/*` | 接口引用 |
| `config/*` | `config/*` | 配置引用 |
| `utils/*` | `app/utils/*` | 工具函数引用 |
| `validator/*` | `app/validator/*` | 校验器引用 |

## 🔧 开发指南

### 常用命令

| 命令 | 说明 |
|-----|------|
| `npm run dev` | 本地开发（egg-bin dev） |
| `npm run sit` | SIT 环境启动（4 workers） |
| `npm run aita-uat` | UAT 环境启动（4 workers） |
| `npm run aita-prod` | 生产环境启动（4 workers） |
| `npm run lint` | ESLint 代码检查 |
| `npm test` | lint + 单元测试 |
| `npm run tsc` | TypeScript 编译 |

### 环境配置

| 环境 | 配置文件 | Nacos group |
|-----|---------|-------------|
| 本地 | `config.local.ts` | chat-service |
| SIT | `config.sit.ts` | chat-service |
| UAT | `config.aita-uat.ts` | chat-service |
| 生产 | `config.aita-prod.ts` | chat-service |
