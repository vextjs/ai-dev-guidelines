# vext 项目

> **项目名称**: vext  
> **项目描述**: 高性能 Node.js Web 框架（支持 Native / Fastify / Hono / Koa / Express 五种 adapter 可替换）

---

## 📂 目录结构

| 目录/文件 | 用途 |
|----------|------|
| [profile/](./profile/) | 🔴 **项目规范**（模块化） |
| [TASK-INDEX.md](./TASK-INDEX.md) | 📋 **任务索引**（追溯所有历史任务） |
| [plans/](./plans/) | 📐 架构方案（v1/v2/v3） |
| [需求.txt](./需求.txt) | 📝 原始需求描述 |
| requirements/ | 需求开发输出（归档） |
| bugs/ | Bug 修复输出（归档） |
| optimizations/ | 优化方案输出（归档） |
| reports/ | 📊 AI 临时报告（gitignore 忽略） |
| .ai-memory/ | 🧠 AI 任务记忆（gitignore 忽略） |
