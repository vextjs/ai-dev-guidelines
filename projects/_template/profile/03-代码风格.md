## 📋 代码风格

### 缩进和格式

- **缩进**: [2 空格 / 4 空格]
- **引号**: [单引号 / 双引号]
- **分号**: [必须 / 不要]

### 命名规范

| 类型 | 规范 | 示例 |
|-----|------|------|
| Controller 文件 | PascalCase | `UserController.ts` |
| Service 文件 | PascalCase | `UserService.ts` |
| Model 文件 | snake_case | `user.ts` |
| 类名 | PascalCase | `UserController` |
| 变量/函数 | camelCase | `getUserById` |
| 常量 | UPPER_SNAKE_CASE | `MAX_RETRY` |
