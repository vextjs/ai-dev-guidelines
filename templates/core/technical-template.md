## ⚙️ 配置设计

### 配置文件结构
```javascript
// config/rate-limit.js
module.exports = {
  // 全局配置
  global: {
    enabled: true,           // 是否启用
    defaultLimit: 100,       // 默认限流阈值
    defaultWindow: 60        // 默认时间窗口（秒）
  },
  
  // API 级别配置
  apis: [
    {
      path: '/api/login',
      limit: 10,
      window: 60
    }
  ]
};
```

### 配置项说明
| 配置项 | 类型 | 默认值 | 说明 |
|-------|------|--------|------|
| enabled | Boolean | true | 是否启用限流 |
| defaultLimit | Number | 100 | 默认限流阈值 |
| defaultWindow | Number | 60 | 时间窗口（秒） |
